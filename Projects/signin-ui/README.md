# Authentication UI - React Native Expo

A modern authentication UI built with React Native Expo featuring:

- Sign Up / Sign In screen
- Forgot Password flow
- Password visibility toggle
- Social login buttons
- Context API state management
- Responsive mobile UI

## Features

### Sign In / Sign Up

- Email input
- Password input
- Confirm password field
- Toggle between Sign In and Sign Up
- Password visibility toggle using eye icons

### Forgot Password

Users can choose different reset methods:

- Email Address
- 2 Factor Authentication
- Google Authenticator

### UI Highlights

- Clean modern design
- Soft shadows and rounded cards
- Lucide React Native icons
- SafeAreaView support
- ScrollView support
- KeyboardAvoidingView support

## Tech Stack

- React Native
- Expo
- Context API
- Lucide React Native
- React Native Safe Area Context
